## Installation

1. Fork and clone this repository to your desktop
2. Navigate to the repo directory
3. Run './bin/ruby_list' 

Download the zip file, 
or in a new terminal window:
```ruby
    $ gh repo clone user_name/rubylist-installation
```
Install yourself as:
```ruby
    $ gem install ruby_list 
```
In case of errors, run
```ruby
    $ bundle exec ruby_list 
```
